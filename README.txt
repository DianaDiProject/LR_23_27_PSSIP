Как пользоваться сайтом

Этот сайт - образовательная платформа для обучения трехмерной графике и моделированию. Здесь можно смотреть уроки и курсы, общаться с другими студентами, создавать свое портфолио и отслеживать прогресс.

Как скачать и собрать архив

Проект выложен на GitHub в разделенном виде, потому что целиком он слишком большой для загрузки. Все файлы нарезаны на части по 20 мегабайт. Чтобы получить рабочий сайт, нужно скачать все эти части и соединить их обратно в один архив.

Сначала зайдите на страницу репозитория на GitHub и скачайте все файлы, которые называются SITE_DONUT_MAIN_part_1, SITE_DONUT_MAIN_part_2, SITE_DONUT_MAIN_part_3 и так далее. Важно скачать все части до последней, иначе архив не соберется. Сохраните все эти файлы в одну папку на вашем компьютере.

Теперь нужно соединить части. Если у вас установлен Git Bash (он обычно ставится вместе с Git), просто нажмите правой кнопкой мыши в папке с файлами и выберите "Git Bash Here". В открывшемся окне введите команду cat SITE_DONUT_MAIN_part_* > SITE_DONUT_MAIN.zip и нажмите Enter. Система соберет все части в один архив.

Если Git Bash у вас нет, можно воспользоваться 7-Zip. Установите программу, нажмите правой кнопкой на файл SITE_DONUT_MAIN_part_aa, выберите 7-Zip, потом "Извлечь файлы". Программа сама найдет остальные части и соберет исходный архив.

После сборки у вас появится файл SITE_DONUT_MAIN.zip. Распакуйте его в удобное место.

Как запустить сайт

Для работы сайта нужен Python. Если он не установлен, скачайте его с официального сайта и при установке обязательно поставьте галочку "Add Python to PATH".

Откройте папку, в которую вы распаковали архив. Нажмите правой кнопкой мыши в пустом месте папки и выберите "Открыть в терминале" или "Git Bash Here".

Теперь нужно подготовить базу данных. Введите команду python manage.py migrate и подождите, пока появятся надписи про успешные миграции.

После этого можно запустить сам сайт. Введите команду python manage.py runserver. Вы увидите сообщение, что сервер запущен по адресу http://127.0.0.1:8000/. Откройте браузер и перейдите по этому адресу.

Как войти на сайт

Когда вы откроете сайт, вы попадете на главную страницу. В правом верхнем углу есть кнопка профиля. Если вы новый пользователь, нажмите на нее, потом выберите "Зарегистрироваться". Заполните форму - придумайте имя, фамилию, логин, email и пароль. После регистрации вернитесь на страницу входа и войдите с теми данными, которые указали.

Для входа в панель администратора нужно добавить в конце адреса /admin. Там можно управлять всем содержимым сайта - добавлять курсы, уроки, смотреть список пользователей. Логин и пароль для входа в админку такие же, как для обычного входа на сайт.

Как пользоваться сайтом

На главной странице можно посмотреть последние уроки и работы студентов. Верхнее меню помогает ориентироваться по сайту. Раздел "Уроки" и "Курсы" - там находятся все учебные материалы. Они разделены по уровням сложности: легкий, средний и сложный. Нажимайте на любой урок или курс, чтобы посмотреть подробности и видео.

Когда вы смотрите урок, внизу есть кнопка "Отметить как пройденный". После этого можно оставить отзыв и добавить свою работу в портфолио. Отзывы помогают другим студентам понять, насколько полезный материал.

Раздел "Сообщество" показывает всех зарегистрированных пользователей. Можно посмотреть, кто чем занимается, и найти единомышленников.

В "Личном профиле" хранится ваша информация. Здесь можно добавить описание, ссылки на соцсети и аватарку. Самое главное - в профиле есть вкладка "Портфолио", куда добавляются все ваши работы. Там же видно, какие курсы и уроки вы уже прошли.

Если возникли вопросы, заходите в раздел "Обратная связь". Там есть ответы на частые вопросы и форма, через которую можно написать администратору.

Что делать, если что-то не работает

Если сайт не запускается, проверьте, установлен ли Python и все ли зависимости установились через pip install. Иногда помогает перезапуск терминала и повторный ввод команды python manage.py runserver.

Если не работает вход, проверьте правильность логина и пароля. Можно попробовать зарегистрироваться заново.

Если видео не грузятся, проверьте интернет-соединение. Видео загружаются с сервиса, так что нужен доступ к этому сервису.

В остальном сайт готов к использованию.
